**foudubus.fr** (Editions du Fou du Bus) est un **blog littéraire** en français par Olivier Garcia : nouvelles courtes, **haïkus**, **micro-nouvelles** et textes de **fiction**.

- **Site** : [https://foudubus.fr](https://foudubus.fr)
- **Contenu** : littérature, nouvelles, haïkus, micro-fiction, chroniques

## À propos du projet

Ce dépôt contient le code source du site **foudubus.fr**. Le site est construit avec [Astro](https://astro.build/) et le thème [ThoughtLite](https://github.com/tuyuritio/astro-theme-thought-lite), déployé sur **Cloudflare**.

| Commande   | Action                          |
| ---------- | ------------------------------- |
| `pnpm dev` | Serveur de développement local  |
| `pnpm build` | Build de production           |
| `pnpm new` | Créer un nouveau contenu (note) |
